◎
ลูกค้า (Accounts)
ทุก Role
เก็บข้อมูล บริษัท / องค์กร ที่เป็นลูกค้า 1 บริษัท = 1 รายการ · Sales จะเห็นเฉพาะลูกค้าที่ตัวเองดูแล
ข้อมูลที่เก็บ
ชื่อบริษัท · ประเภท (บริษัท/ร้านค้า/บุคคล) · โทรศัพท์ · อีเมล · เลขผู้เสียภาษี · พนักงานขายดูแล · ที่อยู่ · ยอดซื้อรวม
การทำงาน
➕ เพิ่มลูกค้า → กรอกข้อมูล → บันทึก
✏️ แก้ไข → กดปุ่ม "แก้ไข" ในแถว
🗑️ ลบ → กดปุ่ม "ลบ" ยืนยันก่อนลบ
🔍 ค้นหา → พิมพ์ในช่องด้านบน กรองตามประเภท
📥 Import → กดปุ่ม "Import Excel" นำเข้าจาก file
💡 เคล็ดลับ: ลูกค้า 1 บริษัท อาจมีผู้ติดต่อหลายคน (ฝ่ายจัดซื้อ, วิศวกร, ผู้จัดการ) ให้บันทึกผู้ติดต่อเหล่านั้นในเมนู "ผู้ติดต่อ" แยกต่างหาก
◑
ผู้ติดต่อ (Contacts)
ทุก Role
เก็บข้อมูล บุคคล ที่ทำงานอยู่ในบริษัทลูกค้า 1 บริษัทมีได้หลายผู้ติดต่อ
ตัวอย่าง: บริษัท ปตท. จำกัด (1 Account)
├── คุณวิภา สมบูรณ์ — วิศวกร (ผู้ติดต่อ 1)
├── คุณอนุชา มีสุข — ฝ่ายจัดซื้อ (ผู้ติดต่อ 2) ← คนออก PO
└── คุณมานพ ดีงาม — ผู้จัดการโรงงาน (ผู้ติดต่อ 3) ← คนอนุมัติ
ข้อมูลที่เก็บ: ชื่อ-นามสกุล · ตำแหน่ง · บริษัทที่สังกัด · แผนก · โทรศัพท์ · อีเมล
วิธีใช้: กด "+ เพิ่มผู้ติดต่อ" → ระบุชื่อบริษัทให้ตรงกับที่บันทึกในหน้าลูกค้า → บันทึก
◷
Activity Log (บันทึกกิจกรรม)
ทุก Role
บันทึกทุกการติดต่อกับลูกค้า ไม่ว่าจะโทร, นัดพบ, ส่งอีเมล หรือโน้ตเตือนความจำ Sales เห็นเฉพาะกิจกรรมของตัวเอง

🤝

✉️

📝
บันทึกโน้ต

💡 เคล็ดลับ: บันทึกกิจกรรมทุกครั้งที่ติดต่อลูกค้า ระบบจะแจ้งเตือนเมื่อไม่ได้ติดต่อนานกว่า 7 วัน ช่วยให้ไม่พลาดการ Follow-up
◻
ใบเสนอราคา (Quotations)
ทุก Role
สร้างและติดตามใบเสนอราคาทั้งหมด รองรับการเชื่อมต่อ FlowAccount เพื่อส่งเอกสารอัตโนมัติ
สถานะใบเสนอราคา
แบบร่างบันทึกไว้ก่อน ยังไม่ส่ง
รอตอบรับส่งให้ลูกค้าแล้ว รอคำตอบ
อนุมัติลูกค้าตอบรับ ถือว่าปิดดีล
ปฏิเสธลูกค้าไม่สนใจ
วิธีสร้างใบเสนอราคา
1️⃣ กด "+ สร้างใบเสนอราคา"
2️⃣ เลือกลูกค้า, พนักงานขาย, วันที่
3️⃣ เพิ่มรายการสินค้า (ระบุจำนวน + ราคา)
4️⃣ ระบบคำนวณ VAT 7% อัตโนมัติ
5️⃣ กด "บันทึก + ส่ง" หรือ "แบบร่าง"
🖨️ พิมพ์ PDF: กดปุ่ม "PDF" ในตาราง → เปิด Preview พร้อมข้อมูลบริษัท Flow Engineering → กด "พิมพ์/บันทึก PDF"
⏰ วันหมดอายุ: หากใบเสนอราคาใกล้หมดอายุ (<3 วัน) วันที่จะแสดงเป็นสีแดง และระบบแจ้งเตือนอัตโนมัติ

จัดการ Users & สิทธิ์
Admin · Manager
เพิ่ม ลบ ปิดใช้งาน User และกำหนดสิทธิ์การเข้าถึงแต่ละเมนู Manager แก้ไขสิทธิ์ได้เฉพาะ Sales, Admin แก้ไขได้ทุก Role
การเพิ่ม User ใหม่
1️⃣ กด "+ เพิ่ม User"
2️⃣ กรอกชื่อ, Username, Password
3️⃣ เลือก Role → สิทธิ์จะตั้งอัตโนมัติ
4️⃣ ปรับสิทธิ์เพิ่มเติมได้ตามต้องการ
5️⃣ กด "บันทึก User"
การจัดการ User ที่มีอยู่
🔴 ปิดใช้งาน → กดปุ่ม "ปิด" Login ไม่ได้ทันที
🟢 เปิดใช้งาน → กดปุ่ม "เปิด"
🗑️ ลบ → เฉพาะ Admin เท่านั้น ลบถาวร
⚠️ ไม่สามารถลบหรือปิดบัญชีตัวเองได้

Audit Log
Admin · Manager
บันทึกทุกการกระทำในระบบ ใช้ตรวจสอบว่าใครทำอะไร เมื่อไหร่ ข้อมูลอะไรถูกแก้ไขหรือลบ
CREATEเพิ่มข้อมูลใหม่ · UPDATEแก้ไขข้อมูล · DELETEลบข้อมูล · LOGINเข้า/ออกระบบ
💡 กรองตาม Action ได้ด้วย dropdown ด้านบนขวา · ข้อมูล Audit บันทึกทั้งใน Browser และ Supabase (ถ้าเชื่อมต่อแล้ว)
📥
Import ข้อมูล (Excel/CSV)
Admin
นำเข้าข้อมูลจำนวนมากจาก Excel หรือ CSV ได้ทีเดียว รองรับ 3 ตาราง
วิธี Import
1️⃣ ไปหน้า "ลูกค้า" → กด "📥 Import Excel"
2️⃣ เลือกประเภท: ลูกค้า / สินค้า / ผู้ติดต่อ
3️⃣ ลากไฟล์วางในกรอบ หรือคลิกเพื่อเลือก
4️⃣ ตรวจสอบตัวอย่าง 5 แถวแรก
5️⃣ กด "บันทึกทั้งหมด"
ดาวน์โหลด Template
กดปุ่ม "📥 ดาวน์โหลด Template" เพื่อรับไฟล์ Excel ตัวอย่างที่มีหัวคอลัมน์ถูกต้องแล้ว กรอกข้อมูลแล้ว Import ได้เลย
⚠️ ต้องใช้ชื่อคอลัมน์ภาษาอังกฤษตามที่กำหนด เช่น name, tel, email
👤
โปรไฟล์ของฉัน
ทุก Role
จัดการข้อมูลส่วนตัวและเปลี่ยนรหัสผ่านของตัวเอง เข้าถึงได้โดยคลิกที่ชื่อตัวเองในแถบด้านซ้าย

เปลี่ยนชื่อที่แสดงในระบบได้ตลอดเวลา ชื่อนี้จะปรากฏใน Activity Log, ใบเสนอราคา และรายงาน

ตั้งค่าระบบ
Admin เท่านั้น
ตั้งค่าการเชื่อมต่อกับ Supabase Database, FlowAccount API และ Line Notify
🗄️ Supabase Database
ใส่ Project URL และ Anon Key จาก supabase.com → Settings → API · กด "ทดสอบ" เพื่อตรวจสอบการเชื่อมต่อ
📊 FlowAccount API
ใส่ Client ID และ Client Secret ที่ได้จาก FlowAccount Developer Portal · ยังไม่มีก็ไม่เป็นไร ใส่ทีหลังได้
📢 Line Notify
ใส่ Token จาก notify-bot.line.me เพื่อรับแจ้งเตือนผ่าน Line เมื่อมีเหตุการณ์สำคัญ
🏢 ข้อมูลบริษัท
ข้อมูลบริษัทที่แสดงในใบเสนอราคา PDF เช่น ชื่อบริษัท เลขภาษี ที่อยู่
